
# Simple Django WebApp

Simple student management system.
*fixed some bugs.


## Run Locally

Clone the project

```bash
  git clone https://github.com/H4K3R13/Django-Sample.git
```

Go to the project directory

```bash
  cd Django-Sample
```

Install Django

```bash
 pip3 install django
```

Start the server

```bash
  python3 manage.py runserver
```
